How to set up VS?

1. To set up the VS first go to the left top corner and click on the last icon which look like 4 cubes

2. after clicking it search for live server and then install it

3. then create a folder on your system and then come back to VS

4. go into file and select open folder and then select the folder which you created

5. then in the left side click bellow the foldername and select new file and name it as index.html

6. after the coding go to file and select save all then see bellow there is a blue line in that there is a option Go live click it

How to print anything on console screen?

To print anything on console screen type the code console.log(type the var name in this);

How to open console screen?

To open the console screen right click on the screen then select inspect and then console
